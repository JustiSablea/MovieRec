Курсовая работа

Гибридная система рекомендаций фильмов

Коллаборативная часть

Item-based сходство рассчитано по оценкам MovieLens. Для фильмов-кандидатов учитывается близость к тем картинам, которые пользователь оценил высоко.

Контентная часть

Профиль строится по жанрам выбранных фильмов. Чем сильнее совпадает жанровый вектор, тем выше content score.

Гибридизация

Итоговый score собирается линейной комбинацией с alpha = 0.7 и небольшой поправкой на качество MovieLens.
